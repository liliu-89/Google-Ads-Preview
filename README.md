# Google-Ads-Preview
This tool makes it possible to generate a preview of Google Ads responsive search ads without having to work in Google Ads itself. It uses HTML, CSS and JavaScript to create and display random combinations of headlines and descriptions. Ideal for presentations, testing and quickly checking ad combinations in a clear preview.
# Google Ads Preview Tool

## Description

This project provides a simple tool that allows you to create a preview of **Google Ads Responsive Search Ads**. The preview shows random combinations of headlines and descriptions that are typically used in Google Ads. The tool is implemented with **HTML**, **CSS** and **JavaScript**, so you can easily customize it and use it in presentations.

## Functions

- Random selection and display of **headlines** and **descriptions**.
- Prevents the display of **duplicate headlines** or **descriptions**.
- **Responsive design** for a clean display on different screen sizes.
- Button** to generate new previews to replicate the dynamics of Google Ads.
- Simple and clear structure that can be easily customized and extended.
